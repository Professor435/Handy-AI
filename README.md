Handy AI — Gesture Intelligence Platform
Created by Mithun AI Lab (@mithun.ai.lab)
A real-time, browser-based hand gesture intelligence platform powered by MediaPipe Hands. Handy AI transforms your webcam feed into an interactive, gesture-controlled experience with three distinct modes — no installation required, runs entirely in the browser.
🚀 Features
Table
Mode	Description	Gestures
▶ Play	Interactive particle physics & lightning effects	Pinch (thumb + index) to spark particles; two hands for lightning bolts & mandala effects; fast movement triggers Matrix rain storm
♥ Shapes	Gesture-recognition shape drawing	Both palms close = animated heart; one-hand peace sign = peace symbol; both fists = star burst explosion
✏ Write	Air drawing / digital whiteboard	Point index finger to draw; pinch to lift pen; shake open palm to clear canvas
